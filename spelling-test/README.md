# IELTS Word Spelling Test - 雅思单词拼写测试系统

## 项目简介

这是一个基于LPM (Learning Progress Monitoring) 研究设计的网页版雅思单词拼写听力测试系统。该系统采用**70%项目复现率**设计，用于课堂中检测学生的单词拼写能力。

## 研究背景

本项目基于以下研究设计：
- **研究框架**: 单组纵向重复测量设计，8周连续测试
- **复现率**: 约70%的单词来自前一次测试，30%为新词
- **理论依据**: 间隔重复效应 (Spacing Effect) 和检索练习 (Testing Effect)

## 功能特点

1. **音频播放**: 点击喇叭图标听单词发音（使用浏览器内置语音合成）
2. **即时反馈**: 提交后立即显示正确/错误，错误时显示正确拼写
3. **自动评分**: 每词5分，总分100分
4. **70%复现率**: 自动从上次测试中选取14个单词复现，6个新词
5. **详细结果**: 测试完成后查看详细答题情况
6. **数据本地存储**: 使用localStorage保存测试历史，支持复现功能

## 使用方法

### 1. 本地测试

直接在浏览器中打开 `index.html` 文件即可使用。

**注意**: 由于使用了localStorage存储数据，建议在Chrome/Edge浏览器中运行以获得最佳兼容性。

### 2. 通过本地服务器运行（推荐）

```bash
# 使用Python启动简单HTTP服务器
cd D:\论文发表\第一篇时间序列LPM word spelling单词测试\单词测试\spelling-test
python -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器访问: `http://localhost:8000`

### 3. 使用Live Server（VS Code）

如果使用VS Code，安装Live Server插件后，右键点击 `index.html` 选择 "Open with Live Server"。

## 测试流程

1. **输入信息**: 输入学号和选择测试次数
2. **开始测试**: 点击"开始测试"按钮
3. **听写单词**: 点击喇叭图标听发音，输入拼写
4. **提交答案**: 按Enter键或点击"提交"按钮
5. **查看反馈**: 系统显示正确/错误及正确拼写
6. **下一题**: 点击"下一个"继续
7. **查看结果**: 20题完成后查看总分和详细报告

## 技术特点

### 70%复现率实现

系统自动从localStorage中读取上次测试的单词列表：
- 如果上次测试 >= 14个单词：随机选择14个复现 + 6个新词
- 如果上次测试 < 14个单词：全部复现 + 补充新词

### 词频控制

词汇库按难度分级（1-5级），确保：
- 排除专有名词、缩写词、极低频词
- 覆盖雅思听力常见话题
- 难度分布合理

### 测量学特性

- 即时反馈：符合形成性评估原则
- 数据追踪：保存完整答题记录
- 可靠性：Cronbach's α ≈ .979（基于研究数据）
