# IELTS Listening Vocabulary Dataset

| 项目 | 数量 |
|------|------|
| 总词条数 | 1,107 |

| 覆盖范围 | 剑4 - 剑20 |

## 📁 文件说明

### 主要文件

- `ielts_vocabulary.json` - JSON格式完整词库
- `总词表.csv` / `总词表.xlsx` - 完整词表
- `统计报告.txt` - 详细的统计分析报告
- `单词_按字母序.csv` - 按字母排序的单词表
- `词组_按字母序.csv` - 按字母排序的词组表

### 按词性分类

- `词性_n_名词.csv` - 名词词表
- `词性_v_动词.csv` - 动词词表
- `词性_adj_形容词.csv` - 形容词词表
- `词性_adv_副词.csv` - 副词词表
- `词性_phr_短语.csv` - 短语词表

### 按难度分类

- `难度_1.csv` - 难度1级词汇（最简单）
- `难度_2.csv` - 难度2级词汇
- `难度_3.csv` - 难度3级词汇
- `难度_4.csv` - 难度4级词汇
- `难度_5.csv` - 难度5级词汇（最难）

## 🔧 数据格式

### JSON格式示例

```json
{
  "word": "absence",
  "pos": "n",
  "difficulty": 3,
  "type": "word"

```

### CSV格式

```csv
序号,单词/词组,词性,类型,难度
1,absence,n,单词,3
```

## 📖 数据来源

本数据集来源于剑桥雅思听力真题（Cambridge IELTS Listening Test）：
- Cambridge IELTS 4-20
- 仅包含听力Section中的填空题答案

## 🎯 使用场景

本数据集适用于：

- 词汇学习与记忆
- 雅思听力备考
- 英语教学研究
- 自然语言处理研究
- 学习分析与教育技术
