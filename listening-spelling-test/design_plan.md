# 雅思听力单词拼写测试网页设计

## 功能需求

### 两种模式

| 功能 | 练习模式 | 测试模式 |
|------|----------|----------|
| 播放次数 | 可多次点击 | 仅1次（自动） |
| 正确答案显示 | 立即显示 | 测试结束后才显示 |
| 立即反馈 | 有 | 无 |
| 进入下一题 | 任意 | 必须回答后 |

### 页面布局

```
┌─────────────────────────────────────────┐
│                                       │
│           [喇叭图标]                 │
│                                       │
│        点击播放单词发音              │
│                                       │
│     ┌──────────────────────┐      │
│     │ [输入框]          │      │
│     └──────────────────────┘      │
│                                       │
│           [确定] [下一个]            │
│                                       │
│     ┌──────────────────────┐      │
│     │ 结果显示区          │      │
│     │  ✓ / ×               │      │
│     │ 正确: apple          │      │
│     └──────────────────────┘      │
│                                       │
│        进度: 5/20                 │
│        得分: 25/100               │
│                                       │
│     [切换模式] [结束测试]            │
└─────────────────────────────────────────┘
```

### 数据结构

```javascript
{
  currentMode: 'practice' | 'test',
  currentWordIndex: 0,
  words: [...],           // 从词库随机抽取20个
  answers: [...],          // 用户输入
  results: [              // 正确/错误
    { word: 'apple', isCorrect: true/false, 
      audioClicks: 0, responseTime: 0 }
  ],
  score: 0,              // 当前得分
  totalScore: 100          // 总分
}
```
